welcome to nyanime!

this is an anime database based off of the much larger and more famous MyAnimeList database. this web application uses the Jikan REST API for all its data.


why was this application made?

MyAnimeList is so full of information that is not really necessary for an average person who is just researching for something new to watch. due to this, nyanime made to just showcase the necessary details in a neater way than MyAnimeList. getting straight to the point, in a more presentable fashion.

notes about this web application

home page: the 'top random genre' chart displays the highest scored animes of a different genre every time the page is loaded.

anime page: if there is a " ¯_(ツ)_/¯ " under any topic, it means there is no info on that specific item from jikan/MyAnimeList.

genres page: genre descriptions were not available through jikan, so they were copied from mynaimelist or made by me.

numbers: scores, ranks, and popularities originate from MyAnimeList and rated by its millions of users.


slow page loads: because jikan has an api rate limit of 3 requests per second, in order to prevent the page from crashing, pauses were implemented so overcome that issue.

nyanime was made with flask and uses JIKAN, an API for MyAnimeList.
